# NoSQL: Social Network API

## Purpose

Utilize a NoSQL database to build an API for a social media website so that it can handle large amounts of unstructured data. 

## Built With
 
 * NoSQL
 * Mongoose ODM
 * MongoDB database
 * Node.js
 * Moment.js
 * Express.js
